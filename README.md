# Graduate Thesis Manuscript Template

## Getting Started in Overleaf

You can use this template in Overleaf by following one of the two methods:

### Method 1: Uploading a ZIP File (Recommended)
1. **Download the Template:** Download the source code as a ZIP archive by clicking [here](https://github.com/ncsu-ne-prag/Graduate_Thesis_Manuscript_Template/archive/refs/tags/1.0.zip).
2. **Create a New Project:** Log in to Overleaf, go to your dashboard, and click the **New Project** button.
3. **Upload the ZIP:** Choose **Upload Project** from the menu and select the ZIP file you just downloaded. Overleaf will automatically extract the files and compile the document.

### Method 2: Importing from GitHub
1. **Fork the Repository:** First, fork the [Template Repository](https://github.com/ncsu-ne-prag/Graduate_Thesis_Manuscript_Template) to your personal GitHub account.
2. **Create a New Project:** In your Overleaf dashboard, click the **New Project** button.
3. **Import the Project:** Select **Import from GitHub**. If you have not linked your GitHub account to Overleaf yet, follow the prompts to authorize the connection.
4. **Choose Your Repository:** Select the fork of the repository from your list of available GitHub projects to import it directly into Overleaf.
